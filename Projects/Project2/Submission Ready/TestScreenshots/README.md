# Test Screenshot Instructions

Save evidence images for each test case listed in TEST_PLAN.md using the exact file names in the Screenshot column.

Recommended capture sequence:
1. Start app and capture launch state.
2. Capture each shape calculation result.
3. Capture torus validation dialog.
4. Capture clear/reset behavior.
5. Capture terminal compile success.

Naming convention examples:
- 01_launch.png
- 02_circle_area.png
- 10_torus_volume.png
- 11_torus_invalid.png
- 17_compile_ok.png
